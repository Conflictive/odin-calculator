# Calculator

A fully functional calculator built for **The Odin Project** foundations curriculum. Features keyboard support and robust error handling.

**[View Live Demo](https://conflictive.github.io/odin-calculator/)**

## Features
* **Keyboard Support:** Works with Numpad, Backspace, and Escape.
* **Error Handling:** Prevents "Divide by Zero" crashes and invalid input.
* **Responsive UI:** Auto-resizes font for long numbers.

## Tech Stack
* JavaScript (ES6+)
* CSS3 (Grid/Flexbox)
* HTML5
